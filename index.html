Technical Overview

Conductor unifies AI providers (OpenAI, Anthropic, Ollama) with a local tool-calling loop. It functions by:

1. Tool Discovery

Each plugin exports a JSON Schema. Upon startup, Conductor registers these capabilities into the LLM system prompt.

2. The Agentic Loop

The agent analyzes user intent, selects a tool, executes it locally, and feeds the output back into the model until the goal is met.

3. Environment Management

All tool execution happens within a defined permission scope. Shell commands require explicit user confirmation by default.
